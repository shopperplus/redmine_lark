## Redmine Lark

实现飞书机器人自动发送issue和回复

### Installation:

Download the plugin and install required gems:

```console
cd /path/to/redmine/plugins
git clone https://github.com/shopperplus/redmine_lark.git
cd /path/to/redmine

```

Restart your app
```console
touch /path/to/redmine/tmp/restart.txt
```
